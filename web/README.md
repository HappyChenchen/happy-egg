# 网页联动台

网页联动台是一个无需构建的静态调试工具，用于从浏览器加入一次性配对房间并发送互动。它不是 macOS 客户端的完整替代品。

## 启动

在仓库根目录运行：

```sh
make web
```

打开 [http://localhost:4173/web/](http://localhost:4173/web/)。

## 使用

1. 在 MacPet 选择“添加好友”→“生成配对码”。
2. 在网页“操作”菜单输入 4 位数字配对码和网页宠物名字。
3. 连接后可发送拍一拍、爱心和庆祝。
4. 网页右上角按钮或宠物右键菜单可调整大小、隐藏宠物或断开网页连接。

默认连接 `wss://happypuppy.io/ws`。本地 relay 可通过查询参数指定：

```text
http://localhost:4173/web/?relay=ws://localhost:8080/ws
```

## 限制

- 网页使用一次性配对房间，不保存 macOS 客户端的长期好友列表。
- 每个配对房间最多两个连接；网页加入时不能已有第二个房间成员。
- 网页身份只保存在当前浏览器的 `localStorage`。
- 这是手动调试工具，目前没有独立自动化测试或生产部署流程。
