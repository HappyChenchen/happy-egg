# MacPet

一个 macOS 桌面宠物互动原型。宠物会悬浮在桌面上；先在右键菜单中按名称配对一位同局域网朋友，再点击宠物发送互动。

## 运行

在本目录执行：

```sh
swift run MacPet
```

需要 macOS 14+ 与 Xcode Command Line Tools。启动后，菜单栏会出现爪印图标；拖动宠物可移动它。

## 当前功能

- 使用 `ai_buddy_assets` 的透明角色素材的桌面宠物窗口
- 左键宠物发送互动；连续快速点击会循环切换不同动作，接收方同步显示同一素材
- 右键宠物打开菜单，按名称配对/取消配对，拍一拍、送爱心、一起庆祝、隐藏或关闭应用
- 右键“宠物大小”可选小（80%）、正常（100%）、大（130%）或超大（160%），选择会被记住
- 菜单栏操作、局域网自动发现、显示/隐藏；互动只会发送给已配对的朋友
- Bonjour + TCP 局域网直连；无需外网或服务端

## 两台 Mac 配对

1. 两台 Mac 连接同一 Wi-Fi 并都启动 MacPet。
2. 在 A 的宠物上右键，打开“配对附近宠物”，选择 B 的 Mac 名称。
3. 菜单出现“已配对：<B 的名称>”后，左键 A 的宠物即可只向 B 发送互动。
4. 两边会显示同一张动作素材；可在右键菜单取消配对并选择其他朋友。

`PetInteractionService` 仍是联机边界，之后可额外实现 Supabase Realtime 版本以支持异地好友。
